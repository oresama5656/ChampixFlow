# ChampixFlow v2.0 💊

チャンピックス（禁煙補助薬）の服用スケジュール・スタンプカード管理システムです。  
薬局での禁煙外来業務をサポートし、患者さんのアドヒアランス向上を支援します。

## 🌟 主な機能

- **患者登録 & スケジュール自動計算**: 服用開始日を入力するだけで、12週間のスケジュールを自動生成します。
- **動的スタンプカード**: 現在の進捗を視覚的に表示し、A5サイズで印刷して患者さんに渡すことができます。
- **スターターキット薬袋印刷**: 初期セット交付時に必要な薬袋デザイン（PDF）をワンクリックで表示・印刷できます。
- **履歴（アーカイブ）管理**: 完了または離脱した患者のデータを履歴として保存・管理できます。
- **ローカル完結型データ管理**: ブラウザの **File System Access API** を使用し、お手元のフォルダに `champix_data.json` としてデータを保存します。サーバーへのアップロードは行われないため、セキュリティ面でも安心です。

## 🚀 セットアップと実行

### システム要件
- Node.js 18 以上

### 起動手順
1. 本プロジェクトのディレクトリをターミナルで開きます。
2. 依存関係をインストールします（初回のみ）：
   ```bash
   npm install
   ```
3. 開発サーバーを起動します：
   ```bash
   npm run dev
   ```
4. ブラウザで `http://localhost:5717` を開きます。
5. 「フォルダを選択」ボタンを押し、データを保存したい任意のローカルフォルダを選択してください。

## 📁 プロジェクト構成

- `client/`: フロントエンド・ソースコード（React + Vite）
  - `src/components/`: 各画面・UIコンポーネント
  - `src/lib/dataManager.js`: データ操作・ロジックの核となるモジュール
  - `public/`: 印刷用PDFなどの静的資産
- `docs/`: 開発経緯や過去の設計ドキュメント
- `package.json`: 依存関係定義
