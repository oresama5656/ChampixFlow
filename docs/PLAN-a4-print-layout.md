# [計画] スタンプカードの印刷レイアウト調整（A5横ベース）とスタンプの重ね表示実装

印刷レイアウトをA5横（210mm x 148.5mm）サイズに固定し、A4用紙（縦）の上半分に配置されるように調整します。また、記録済みの「チェックマーク」を、元の文字（用量など）を消さずにその上に重ねて表示（オーバーレイ）させることで、実際のスタンプのような視覚効果を実現します。

## 成功の基準
- [ ] 印刷サイズが A5横（210mm x 148.5mm）で固定されており、A4縦で印刷した際に上半分に配置される。
- [ ] スタンプ（済マーク）がセルの元のテキスト（用量など）を隠さず、その上に重ねて表示される。
- [ ] タイポグラフィと余白が改善され、モダンで高級感のあるデザインになっている。
- [ ] 背景グラフィックがオフでも、重ね表示の視認性が保たれている。

## 変更予定のファイル
- [ ] `client/index.css`: `@page` 設定、A5コンテナの固定、および「重ねスタンプ（Overlay Check）」用のCSS定義。
- [ ] `client/src/components/StampCard.jsx`: セル内要素の重なり構造（Relative/Absolute）の実装、デザインのブラッシュアップ。
